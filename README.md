dDMC
====
Dispersion correction tailored around the SCC-DFTB approach
Look in src/main.f90 for more info!


HOW TO COMPILE
--------------
You need a recend gfortran compilator installed on you machine.

$ cd src
$ make

Tested on Debian 8
